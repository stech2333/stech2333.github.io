Git 简易教程

基本介绍

Git 是目前世界上最先进的分布式版本控制系统，我们将用它来管理 PA 课设的版本更迭。

常见概念

工作区：在你的计算机上能看到的目录。

版本库：又名仓库，是新建版本库时在工作区下生成的一个隐藏目录.git（通过 ls -a 命令可以看到）。

暂存区：在 Git的版本库中，需要提交的文件修改通通放到暂存区，然后一次性提交暂存区的所有修改 。
